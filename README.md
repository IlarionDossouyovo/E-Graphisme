# E-Graphisme & E-Studio By ELECTRON

Plateforme de design graphique et production vidéo nouvelle génération.

## 🚀 Fonctionnalités

### E-Graphisme (Design Graphique & Web)
- **Design Responsive** - Compatible mobile, tablette et desktop
- **Navigation Smooth** - Défilement fluide entre les sections
- **Portfolio Interactif** - Filtrage des projets par catégorie
- **Formulaire de Contact** - Validation en temps réel
- **Animations Modernes** - Effets visuels et transitions fluides
- **Dark/Light Mode** - Thème sombre et clair
- **Chat en Direct** - Widget WhatsApp, callback, email
- **Blog Dynamique** - Système de blog avec commentaires
- **Panneau Admin** - Gestion des messages et du contenu
- **SEO Optimisé** - Meta tags, JSON-LD, sitemap
- **Sécurité** - CSRF, Rate Limiting, CAPTCHA

### E-Studio (Production Vidéo IA)
- **Logo Premium** - Design futuriste et technologique
- **Interface Futuriste** - UI sombre avec effets néon
- **Fonctionnalités IA** - Montage intelligent
- **Page d'atterrissage** - Design SaaS premium

## 🛠️ Technologies

- **HTML5** - Structure sémantique
- **CSS3** - Design moderne avec variables et animations
- **JavaScript** - Interactivité
- **PHP** - Backend et API
- **SVG** - Logos vectoriels
- **Font Awesome** - Icônes
- **Google Fonts** - Orbitron, Inter, Poppins, Playfair Display

## 📁 Structure du Projet

```
E-Graphisme/
├── index.html              # Page principale E-Graphisme
├── studio.html            # Page E-Studio (Production Vidéo IA)
├── blog.php               # Blog dynamique
├── sitemap-generator.php  # Générateur de sitemap XML
├── images/
│   ├── logo-estudio.svg   # Logo E-Studio complet
│   └── logo-estudio-icon.svg # Icône E-Studio
├── admin/                 # Panneau administrateur
├── php/                   # Backend
│   ├── contact.php        # Formulaire sécurisé
│   ├── comments.php       # Système de commentaires
│   ├── api.php           # API REST
│   ├── upload.php        # Upload sécurisé
│   └── cache.php         # Système de cache
├── js/
│   ├── main.js           # Scripts principaux
│   ├── chat-widget.js    # Chat en direct
│   └── toast.js         # Notifications toast
├── css/
│   ├── style.css
│   ├── pages.css
│   └── extra.css
├── uploads/               # Fichiers uploadés
├── cache/                 # Cache
└── en/                   # Version anglaise
```

## 🔒 Sécurité

- Protection CSRF sur tous les formulaires
- Rate Limiting (5 requêtes/minute)
- Time-based CAPTCHA anti-bot
- Validation et sanitation des entrées
- Upload sécurisé avec validation MIME

## 🚦 Installation

```bash
# Cloner le dépôt
git clone https://github.com/IlarionDossouyovo/E-Graphisme.git

# Ouvrir le dossier
cd E-Graphisme

# Lancer un serveur local
python3 -m http.server 8080
```
