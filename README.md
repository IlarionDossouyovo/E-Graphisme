# E-Graphisme 🎨

Site web professionnel pour une agence de design graphique et web.

## 🚀 Fonctionnalités

- **Design Responsive** - Compatible mobile, tablette et desktop
- **Navigation Smooth** - Défilement fluide entre les sections
- **Portfolio Interactif** - Filtrage des projets par catégorie
- **Formulaire de Contact** - Validation en temps réel
- **Animations Modernes** - Effets visuels et transitions fluides

## 📁 Structure

```
E-Graphisme/
├── index.html          # Page principale
├── css/
│   └── style.css       # Styles CSS
├── js/
│   └── main.js         # Scripts JavaScript
└── README.md           # Documentation
```

## 🛠️ Technologies

- **HTML5** - Structure sémantique
- **CSS3** - Design moderne avec variables et animations
- **JavaScript** - Interactivité
- **Font Awesome** - Icônes
- **Google Fonts** - Typographie (Poppins, Playfair Display)

## 🚦 Installation

```bash
# Cloner le dépôt
git clone https://github.com/IlarionDossouyovo/E-Graphisme.git

# Ouvrir le dossier
cd E-Graphisme

# Lancer un serveur local
python3 -m http.server 8080
```
